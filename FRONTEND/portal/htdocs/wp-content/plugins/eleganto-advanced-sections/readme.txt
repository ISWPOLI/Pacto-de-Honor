Add Services, Testimonials, Slider, Image, Blog and Contact sections to Eleganto theme.

== Description ==

Eleganto Advanced Sections creates advanced sections for Eleganto WordPress theme by themes4wp.com. Add Services, Testimonials, Slider, Image, Blog and Contact sections to Eleganto theme. This plugin won't do anything for other free or premium WordPress themes and you need to download and install <a href="https://wordpress.org/themes/eleganto/" target="_blank">Eleganto</a>.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the whole contents of the folder `eleganto-advanced-sections` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress dashboard
3. Enjoy using it :)

== Frequently Asked Questions ==

= What themes this plugin supports? =

Currently it works only with Eleganto theme.

= Am I obligated to use it? =

You can still use Eleganto theme without this plugin but you won't be able to setup homepage sections.
