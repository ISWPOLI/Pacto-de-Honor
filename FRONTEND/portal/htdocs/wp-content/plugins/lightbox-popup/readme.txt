== Installation ==

### First of all download our WordPress Lightbox plugin ZIP file.
  
Follow these steps for installing Lightbox plugin

1. Log in to your WordPress website admin panel.   
2. Then open the Plugins page and click Add New > Upload Plugin.    
3. Then click "Choose file" then select our [WordPress Lightbox plugin](http://wpdevart.com/wordpress-lightbox-plugin/) zip file. 
4. Install and activate the plugin.     
5. Now go to our plugin settings page(from the left panel of Dashboard) and configure Lightbox options.

Now your images will be displayed in lightbox. If any problem occurs contact us at support@wpdevart.com.

== Frequently Asked Questions ==

Our WordPress Lightbox is an user friendly plugin, but here you can check some frequently asked questions.

= What files I need to upload for installing the Lightbox Popup plugin =

You need to select the .zip file only, there is no need to extract the zip file, just upload it.

= Does your lightbox support videos =

Yes dear uses, it supports YouTube and Vimeo videos.

= Can I display images and videos in the same lightbox  =

Yes, it's possible as well.

= I purchased the Lightbox Popup pro, but didn't download the pro file =

Contact us at support@wpdevart.com and we will send you the pro version.

= How can I add video lightbox =

You need to open your post/page editor and type the video url(YouTube, Vimeo) and choose popup image.
Here is an example.

`<a href="https://www.youtube.com/watch?v=mZb_gat5YCY"><img src="http://demo.wpdevart.com/wp-content/uploads/2015/12/121-1024x576.jpg" alt="121" width="474" height="267" class="alignnone size-large wp-image-127" /></a>`

= Can I use lightbox shortcode for images on widget =

Yes dear users, you can use rel="wpdevart_lightbox" attribute.

= I'm using other gallery plugin and I need to display your lightbox for that gallery  =

Unfortunately almost every gallery plugin have his own lightbox addon, so we can't help in this case.

= How should I activate the Lightbox Popup =

You just need to install our plugin and activate it, then Enable it from admin menu and add images, that's all.

= Do I need to pay yearly for Lightbox pro version =

No dear users, you need to pay only once for our Lightbox Popup pro version.
