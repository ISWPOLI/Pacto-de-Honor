creditos.js


En esta clase el jugador podrá observar todo lo relacionado al pacto de honor, como por ejemplo que es el pacto de honor, la descripción de personalidad de los personajes bueno, la descripción de personalidad de los personajes malos y la descripción de cada uno de los 8 tipos de plagio.



Funciones
Dato de retorno Descripción breve
void preload ( )
Función de Phaser, que sirve para cargar cosas.
void create ( )
Función de Phaser, que sirve para agregar cosas.
void verPacto ( )
Mostrará en la ventana del juego, que es el pacto de honor.
void verPersonajesBuenos ( )
Mostrará en la ventana del juego la descripción de personalidad de todos los personajes buenos.
void verPersonajesMalos ( )
Mostrará en la ventana del juego la descripción de personalidad de todos los personajes malos.
void verPlagios ( )
Mostrará en la ventana del juego la descripción de cada tipo de plagio.
void volver ( )
Regresará a la sección del juego llamada navegación después de que el botón regresar sea presionado.


preload( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser en la que cargaremos todos los componentes a ser utilizados en el juego como imágenes, sonidos, fondos, sprites, etc. Mas información aqui. No tiene retorno.

create( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser, aquí se añadirán los componentes al juego. En esta función asignamos la posición inicial, el tamaño, etc. De todos los componentes. Mas información aqui. No tiene retorno.

verPacto( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función mostrará al jugador que es el pacto de honor en la ventana de juego. Retorna una descripción sobre que es el pacto del honor.

verPersonajesBuenos( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función mostrará al jugador la descripción de cada personaje bueno en la ventana de juego. Retorna una descripción de personalidad sobre cada personaje bueno.

verPersonajesMalos( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función mostrará al jugador la descripción de cada personaje malo en la ventana de juego. Retorna una descripción de personalidad sobre cada personaje malo.

verPlagios( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función mostrará al jugador la descripción de cada uno de los 10 tipos de plagios en la ventana de juego. Retorna una descripción de cada tipo de plagio.

volver( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función permitirá que se regrese a la sección llamada navegacion.js después de que el botón volver sea presionado. No tiene retorno.
