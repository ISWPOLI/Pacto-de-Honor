VerificacionDeCodigo.js


Esta clase se usará para validar un código alfanúmerico introducido por el jugador, en la cual se valida que tenga 11 caracteres, que empiece con un número del 1-9, que en la sexta posición alla un guión y que en la ultima posición alla una "P" o una "E".



Funciones
String validarString (String cadenaAnalizar)
Verificar si el código alfanúmerico es valido.


validarString
Esta función recibe como parametro una cadena cadenaAnalizar que representa un código alfanúmerico.

Verificará el código alfanúmerico introducido, y determinará si este es valido o no.

Retornará una cadena especificando si el código alfanúmerico esta mal escrito o no existe.


Variables
mapnumeros1 boolean [ ] Arreglo de 9 posiciones para identificar cada uno de los personajes que se desbloquea
string String Cadena que contiene todas las letras y números del abecedario
mapletras1 boolean [ ] Arreglo de 36 posiciones, en el cual cada posición representa una posición de "ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789"
