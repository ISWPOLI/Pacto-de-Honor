cajaMisteriosa.js


En esta clase se muestran los 9 tipos de cajas sorpresa que hay, mostrando al jugador las caracteristicas y recompensas de cada una de ellas.



Funciones
void preload ( )
Función de Phaser, que sirve para cargar cosas.
void create ( )
Función de Phaser, que sirve para agregar cosas.
void verCaja ( )
Se dirigirá a la seccion de navegacion.
void update ( )
Función de Phaser, que sirve para cambiar cosas en el juego.


preload( )
No recibe parámetros.

Esta es una función del framework Phaser en la que cargaremos todos los componentes a ser utilizados en el juego como imágenes, sonidos, fondos, sprites, etc. Mas información aqui.

No tiene retorno.

create( )
No recibe parámetros.

Esta es una función del framework Phaser, aquí se añadirán los componentes al juego. En esta función asignamos la posición inicial, el tamaño, etc. De todos los componentes. Mas información aqui.

No tiene retorno.

verCaja( )
No recibe parámetros.

Esta función se encargará de dirigir el juego a la sección navegacion.js

No tiene retorno.

update( )
No recibe parámetros.

Esta es una función del framework Phaser, aquí estarán todos los cambios o ‘acciones’ del juego. Qué hacer al presionar una tecla, al mover el cursor, al chocar una pared; en fin, toda la lógica del juego. Mas información aqui.

No tiene retorno.


Variables
speedMult Double Representa la velocidad dentro de un scroll.
friction Double Representa la friccion dentro del scroll.
apodo String Cadena que representa el nombre de una caja sorpresa en un determinado momento dentro de un scroll.
boxloocked Boolean [ ] Arreglo de 9 posiciones, las cuales representan si una caja sorpresa esta bloqeuada o desbloqueada.
description String Cadena que representa la descripción de una caja sorpresa en un determinado momento dentro de un scroll.
startButton PhaserButton Botón que muestra al jugador si una caja sorpresa esta bloqueada o desbloqueada.
zoomed Boolean Representa si una imagen dentro del scroll esta seleccionada o no.
