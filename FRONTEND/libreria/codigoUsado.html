codigoUsado.js


Esta clase mostrará dinamicamente en pantalla al jugador que el código alfanúmerico que esta introduciendo ya fue usado.



Funciones
void preload ( )
Función de Phaser, que sirve para cargar cosas.
void create ( )
Función de Phaser, que sirve para agregar cosas.
String nextLine ( )
Devuelve un texto que esta en una posición de un arreglo.
String nextWord ( )
Devuelve una palabra de un texto que esta en una posición de un arreglo.


preload( )
No recibe parámetros.

Esta es una función del framework Phaser en la que cargaremos todos los componentes a ser utilizados en el juego como imágenes, sonidos, fondos, sprites, etc. Mas información aqui.

No tiene retorno.

create( )
No recibe parámetros.

Esta es una función del framework Phaser, aquí se añadirán los componentes al juego. En esta función asignamos la posición inicial, el tamaño, etc. De todos los componentes. Mas información aqui.

No tiene retorno.

nextLine( )
No recibe parámetros.

Con esta función se obtendra un texto que esta en una posición de un arreglo de String.

Retorna un string que representa un texto que se mostrará al jugador.

nextWord( )
No recibe parámetros.

Obtiene la cadena de texto devuelta por nextLine( ) y la recorre mostrando al jugador palabra por palabra de esta cadena.

Retorna un String que representa una palabra que se mostrará al jugador en la ventana.


Variables
contenido String [ ] Arreglo de 3 posiciones que contiene 3 cadenas de texto que se mostraran al jugador, las cuales son "Solo puedes usar un código de regalo y..", "Este ya lo has usado" y "¡Lo sentimos!".
line String Cadena de texto que contiene una de las 3 posiciones de contenido[ ].
wordIndex int Entero que representa el indice de la palabra en que se esta
lineIndex int Entero que representa el indice dentro del arreglo contenido[ ].
wordDelay int Entero que representa el tiempo en milisegundos en que se repetirá la función nextLine( ).
lineDelay int Entero que representa el tiempo en milisegundos en que se repetirá la función nextWord( ).
regalado PhaserGame Variable para inicializar a librería de Phaser.
button PhaserButton Botón con el que agregará un botón de regresar a la sección anterior.
text PhaserText String que se agregará a la ventana de juego, representando el texto que contiene contenido[ ].
