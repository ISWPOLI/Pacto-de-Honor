desbloqueoPersonaje.js


Esta clase se usa para obtener o regalar un personaje del juego mediante un código alfanúmerico, cuando se obtiene un personaje se muestra al jugador si fue regalado por un estudiante o por un profesor.

preload( )
No recibe parámetros.

Esta es una función del framework Phaser en la que cargaremos todos los componentes a ser utilizados en el juego como imágenes, sonidos, fondos, sprites, etc. Mas información aqui.

No tiene retorno.

create( )
No recibe parámetros.

Esta es una función del framework Phaser, aquí se añadirán los componentes al juego. En esta función asignamos la posición inicial, el tamaño, etc. De todos los componentes. Mas información aqui.

No tiene retorno.

validarString( )
No recibe parámetros.

Verificará el código alfanúmerico introducido, y determinará si este es valido o no.

Retornará una cadena especificando si el código alfanúmerico esta mal escrito o no existe y si fue regalado por un profesor o por un estudiante.

verPerfilJugador( )
No recibe parámetros.

No tiene retorno
