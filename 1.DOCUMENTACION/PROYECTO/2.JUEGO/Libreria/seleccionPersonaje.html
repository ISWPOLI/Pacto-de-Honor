preload( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser en la que cargaremos todos los componentes a ser utilizados en el juego como imágenes, sonidos, fondos, sprites, etc. Mas información aqui. No tiene retorno.

create( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser, aquí se añadirán los componentes al juego. En esta función asignamos la posición inicial, el tamaño, etc. De todos los componentes. Mas información aqui. No tiene retorno.

verBatalla( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función verifica que el jugador haya seleccionado algún personaje, en caso de no ser asi mostrará un texto en la ventana, el cual es "Selecciona un personaje". Después de seleccionarlo y presionar el botón seleccionar, iniciará la sección llamada batalla.js. No tiene retorno.
