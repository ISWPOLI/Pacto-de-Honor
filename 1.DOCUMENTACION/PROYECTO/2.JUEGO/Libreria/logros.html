logros.js


En esta clase se muestran al jugador los tipos de logros que hay, su nombre y como desbloquearlos. Además cada logro mostrará su estado (bloqueado o desbloqueado).

preload( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser en la que cargaremos todos los componentes a ser utilizados en el juego como imágenes, sonidos, fondos, sprites, etc. Mas información aqui. No tiene retorno.

create( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser, aquí se añadirán los componentes al juego. En esta función asignamos la posición inicial, el tamaño, etc. De todos los componentes. Mas información aqui. No tiene retorno.

isUnloocked( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función verifica si un logro esta bloqueado o desbloqueado. No tiene retorno.

playedUnloock( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función bloquea todos los logros. No tiene retorno.

recompensa(int exp, int monedas)
Parámetros Descripción Retorno
Recibe como parámetros los puntos de experiencia y la cantidad de monedas que se obsequieran al jugador. Esta función da recompensas economicas y de experiencia al jugador. No tiene retorno.

isMikuplagio(int i)
Parámetros Descripción Retorno
Recibe como parámetro un índice númerico que representara una posición en el arreglo de desbloqueo de logros. Esta función verifica que el jugador lleve por lo menos 100 horas de juego. No tiene retorno.

isMiLifePlagio(int i)
Parámetros Descripción Retorno
Recibe como parámetro un índice númerico que representara una posición en el arreglo de desbloqueo de logros. Esta función verifica si el jugador esta dentro del top 5 del ranking. No tiene retorno.

isMiSkillPlagio(int i)

isTheOriginalPlagio(int i)

isMiHitckGround(int i)
Parámetros Descripción Retorno

isTheUnloocker(int i)

verPerfil( )

update( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser, aquí estarán todos los cambios o ‘acciones’ del juego. Qué hacer al presionar una tecla, al mover el cursor, al chocar una pared; en fin, toda la lógica del juego. Mas información aqui. No tiene retorno.
