seleccionAvatar.js


En esta clase el jugador seleccionará un personaje inicial con el cual empezará el juego.



Funciones
Dato de retorno Descripción breve
void create ( )
Función de Phaser, que sirve para agregar componentes al juego.
void verH ( )
Se dirigirá a la seccion llamada historieta.js.
void update ( )
Función de Phaser, que sirve para cambiar componentes en el juego.


create( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser, aquí se añadirán los componentes al juego. En esta función asignamos la posición inicial, el tamaño, etc. De todos los componentes. Mas información aqui. No tiene retorno.

verH( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función se encargará de dirigir el juego a la sección historieta.js. No tiene retorno.

update( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser, aquí estarán todos los cambios o ‘acciones’ del juego. Qué hacer al presionar una tecla, al mover el cursor, al chocar una pared; en fin, toda la lógica del juego. Mas información aqui. No tiene retorno.

namesCharacters String [ ] Arreglo de 9 posiciones que contiene el nombre de cada uno de los 9 personajes buenos.
description String Cadena que representará la descripción de personalidad de cada uno de los 9 personajes buenos en un determinado momento del scroll.
apodo String Cadena que representará el nombre de cada uno de los 9 personajes buenos en un determinado momento del scroll.
music PhaserButton Botón que al ser presionado reproducirá un audio.
startButton PhaserButton Botón que al ser presionado iniciará la sección llamada historieta.js.
avatarSeleccionado String Cadena que contiene el id del avatar seleccionado.
descriptions String [ ] Arreglo de 9 posiciones que contiene las descripciones de personalidad de los personajes buenos.
