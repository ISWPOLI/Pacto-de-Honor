VerificacionDeCodigo.js


Esta clase se usará para validar un código alfanúmerico introducido por el jugador, en la cual se valida que tenga 11 caracteres, que empiece con un número del 1-9, que en la sexta posición alla un guión y que en la ultima posición alla una "P" o una "E".



Funciones
Dato de retorno Descripción breve
String validarString (String cadenaAnalizar)
Verificar si el código alfanúmerico es valido.


validarString
Parámetros Descripción Retorno
Esta función recibe como parametro una cadena cadenaAnalizar que representa un código alfanúmerico. Verificará el código alfanúmerico introducido, y determinará si este es valido o no. Retornará una cadena especificando si el código alfanúmerico esta mal escrito o no existe.
