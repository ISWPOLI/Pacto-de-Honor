movil.js


En esta clase se detecta si el jugador esta accediendo al juego desde Android, BlackBerry, IOS, Opera o Windows.



Funciones
Dato de retorno Descripción breve
String Android ( )
Verifica si el jugador accede al juego desde un dispositivo android.
String BlackBerry ( )
Verifica si el jugador accede al juego desde un dispositivo BlackBerry.
String iOS ( )
Verifica si el jugador accede al juego desde un dispositivo IOS.
String Opera ( )
Verifica si el jugador accede al juego desde un dispositivo con Opera.
String Windows ( )
Verifica si el jugador accede al juego desde un dispositivo Windows.
String any ( )
Devuelve cualquiera dispositivo de acceso.


Android( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función verifica si el jugador accede al juego desde un dispositivo Android. Retorna una cadena especificando que el dispositivo del que se accede al juego es Android.
BlackBerry( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función verifica si el jugador accede al juego desde un dispositivo BlackBerry. Retorna una cadena especificando que el dispositivo del que se accede al juego es BlackBerry.
iOS( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función verifica si el jugador accede al juego desde un dispositivo IOS. Retorna una cadena especificando que el dispositivo del que se accede al juego es IOS.
Opera( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función verifica si el jugador accede al juego desde un dispositivo con Opera. Retorna una cadena especificando que el navegador del que se accede al juego es Opera.
Windows( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función verifica si el jugador accede al juego desde un dispositivo Windows. Retorna una cadena especificando que el dispositivo del que se accede al juego es Windows.
any( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función devuelve cualquier dispositivo de acceso. Retorna una cadena con cualquier dispositivo de acceso.
