batalla.js


Dependiendo del dispositivo del cual se accede al juego se agregan las barras de vida y energia del jugador y de la computadora, contiene la logica de los golpes, impactos de cada uno y el tiempo de la batalla.



Funciones
Dato de retorno Descripción breve
void preload ( )
Función de Phaser, que sirve para cargar componentes del juego.
void create ( )
Función de Phaser, que sirve para agregar componentes al juego.
void Render ( )
Función de Phaser, que sirve renderizar el juego.
void update ( )
Función de Phaser, que sirve para cambiar componentes en el juego.


preload( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser en la que cargaremos todos los componentes a ser utilizados en el juego como imágenes, sonidos, fondos, sprites, etc. Mas información aqui. No tiene retorno.

create( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser, aquí se añadirán los componentes al juego. En esta función asignamos la posición inicial, el tamaño, etc. De todos los componentes. Mas información aqui. No tiene retorno.

render( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser, aquí se renderizará el juego. Mas información aqui. No tiene retorno.

update( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser, aquí estarán todos los cambios o ‘acciones’ del juego. Qué hacer al presionar una tecla, al mover el cursor, al chocar una pared; en fin, toda la lógica del juego. Mas información aqui. No tiene retorno.


Variables
Nombre Tipo de dato Explicación
idNivel int Representa el id del nivel seleccionado.
personajeJugador PhaserSprite Sprite del personaje del jugador.
personajeComputadora PhaserSprite Sprite del personaje de la computadora.
cursores PhaserKeyboard Cursores que se usaran para que el jugador mueva al personaje.
counter int Representa el tiempo de la partida.
ti int Muestra el avance del tiempo de la partida en segundos.
danoH Double [ ] Arreglo de 3 posiciones que indica el daño que ocasiona el jugador con cada uno de sus ataques.
danoV Double [ ] Arreglo de 3 posiciones que indica el daño que ocasiona la computadora con cada uno de sus ataques.
musicButton PhaserButton Botón que se usa para agregar sonidos a la batalla.
movH Boolean [ ] Arreglo de 4 posiciones, las cuales, la primera significa que el jugador se esta defendiendo, la segunda significa que el jugador esta atacando con el ataque normal, la tercera significa que el jugador esta atacando con el ataque de personalidad y la cuarta posición significa que el jugador se esta defendiendo de un ataque tipo plagio.
movV Boolean [ ] Arreglo de 4 posiciones, las cuales, la primera significa que la computadora se esta defendiendo, la segunda significa que la computadora esta atacando con el ataque normal, la tercera significa que la computadora esta atacando con el ataque de personalidad y la cuarta posición significa que la computadora esta atacando con el ataque tipo plagio.
ataquePlagio PhaserSprite Sprite del poder de plagio de la computadora.
ataquePersonalidadC PhaserSprite Sprite del poder de personalidad de la computadora.
idPJ int Entero que contiene el id del personaje del jugador
idPC int Entero que contiene el id del personaje la computadora
costoAtaqueJ double Costo en la barra de energía cuando el jugador use el ataque de personalidad
costoAtaqueC double Costo en la barra de energía cuando la computadora use el ataque de personalidad
costoPlagioC double Costo en la barra de energía cuando la computadora use el ataque dtipo plagio
secuencia boolean variable booleana que esta en tru cuando la computadora este realizando un movimiento y en false cuando no lo este realizando.
movimientoComputadora String Cadena que identificará el movimiento que realizará la computadora.
primerImpacto Boolean Se usa para que el ataque tipo plagio de la computadora solo reste vida una vez.
indice int Indice que representa el indice del movimiendo que esta realizando la computadora.
fondoGame PhaserSprite Sprite del fondo de la batalla.
ataquePersonalidadJ PhaserSprite Sprite del ataque de personalidad del jugador.
escudo1 PhaserSprite Sprite del primer escudo del jugador.
escudo2 PhaserSprite Sprite del segundo escudo del jugador.
gamePad PhaserPlugin Plugin de phaser para utilizar joystick en los dispositivos moviles.
joystick PhaserJoystick Joystick para queel jugador mueva al personaje durante la batalla.
pausa PhaserButton Botón para pausar la batalla.
