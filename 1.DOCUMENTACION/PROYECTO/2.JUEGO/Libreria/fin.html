fin.js


En esta clase se obtiene el ganador de la batalla y se muestra su nombre y sus recompensas.



Funciones
Dato de retorno Descripción breve
void preload ( )
Función de Phaser, que sirve para cargar componentes del juego.

Función de Phaser, que sirve para agregar componentes al juego.
void volver ( )
Se dirigirá a la seccion llamada navegacion.js.


preload( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser en la que cargaremos todos los componentes a ser utilizados en el juego como imágenes, sonidos, fondos, sprites, etc. Mas información aqui. No tiene retorno.

create( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser, aquí se añadirán los componentes al juego. En esta función asignamos la posición inicial, el tamaño, etc. De todos los componentes. Mas información aqui. No tiene retorno.

volver( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función se encargará de dirigir el juego a la sección navegacion.js. No tiene retorno.


Variables
Nombre Tipo de dato Explicación
resultado String Cadena que contiene el resultado del jugador en la batalla, esta cadena puede ser "GANASTE" o "PERDISTE".
name String Cadena que representa el nombre del ganador.
volver PhaserButton Botón que al ser presionado, regresará a la sección navegacion.js.
monedas String Monedas ganadas por la batalla.
experiencia String Experiencia ganada por la batalla.
