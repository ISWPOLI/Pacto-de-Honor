rankings.js


En esta clase se clasifican a los jugadores del juego dependiendo de diferentes tipos de ranking, los cuales son diario, semanal, general y a nivel del Politecnico Grancolombiano.

preload( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser en la que cargaremos todos los componentes a ser utilizados en el juego como imágenes, sonidos, fondos, sprites, etc. Mas información aqui. No tiene retorno.

create( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser, aquí se añadirán los componentes al juego. En esta función asignamos la posición inicial, el tamaño, etc. De todos los componentes. Mas información aqui. No tiene retorno.

volver( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función se encargará de dirigir el juego a la sección navegacion.js. No tiene retorno.

verDiario( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función mostrará el ranking por día. No tiene retorno.

verSemanal( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función mostrará el ranking por semana. No tiene retorno.

verPoli( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función mostrará el ranking a nivel del Politecnico Grancolombiano. No tiene retorno.

verGeneral( )
Parámetros Descripción Retorno
No recibe parámetros. Esta función mostrará el ranking general. No tiene retorno.
