desbloqueoPersonaje.js


Esta clase se usa para obtener o regalar un personaje del juego mediante un código alfanúmerico, cuando se obtiene un personaje se muestra al jugador si fue regalado por un estudiante o por un profesor.

preload( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser en la que cargaremos todos los componentes a ser utilizados en el juego como imágenes, sonidos, fondos, sprites, etc. Mas información aqui. No tiene retorno.

create( )
Parámetros Descripción Retorno
No recibe parámetros. Esta es una función del framework Phaser en la que cargaremos todos los componentes a ser utilizados en el juego como imágenes, sonidos, fondos, sprites, etc. Mas información aqui. No tiene retorno.

validarString( )
Parámetros Descripción Retorno
No recibe parámetros. Verificará el código alfanúmerico introducido, y determinará si este es valido o no. Retornará una cadena especificando si el código alfanúmerico esta mal escrito o no existe y si fue regalado por un profesor o por un estudiante.

verPerfilJugador( )
Parámetros Descripción Retorno
No recibe parámetros. Se dirige a la sección del juego llamada perfilJugador.js. No tiene retorno.
