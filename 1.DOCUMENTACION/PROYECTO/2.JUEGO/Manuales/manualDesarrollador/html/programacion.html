Programación Poli Fighters


FRONTEND



HTML + CSS + Javascript



El juego está programado en lenguaje Javascript, utiliza HTML en su versión 5.0 y Hojas de Estilo en Cascada CSS3.

La página oficial de Javascript es esta

Framework Phaser



El juego utiliza un Framework llamado Phaser, que contiene funciones que se requieren a lo largo del juego. De esta manera no hay que programar todas las funciones desde cero.

Puedes visitar la página oficial de Phaser aquí

BACKEND



Base de Datos MySQL



Poli Fighters utiliza una base de datos MySQL para todo el manejo de datos del juego, de los usuarios y de las pruebas psicotécnicas.

Ve a la página oficial de MySQL en este enlace

Servicios REST



Poli Fighters incluye servicios web de tipo REST para la comunicación entre el juego y la base de datos. Asì se puede consultar o insertar en la base de datos.
