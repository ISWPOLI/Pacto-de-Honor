/1.DOCUMENTACION



Contiene todos la documentación realizada por los Scrum Master, además de documentación de administración y requerimientos.


También están las guías sobre el juego, la librería y el manual de usuario y de desarrollador.

/BACKEND



En esta carpeta están todos los archivos que se requieren para el Backend del juego.

/FRONTEND



En la carpeta Frontend están las carpetas de Juego y Portal.


Aquí verás la descripción de cada carpeta con sus subcarpetas.

/FRONTEND/juego



Esta es la carpeta principal del juego, contiene las carpetas CSS, HTML, img, java, js y JSP.


A continuación verás con detalle cada carpeta.

/FRONTEND/juego/css



Contiene los archivos CSS que le dan el estilo al juego en su versión web y móvil.

/FRONTEND/juego/html



Contiene el archivo html principal del juego.


Este archivo carga todos los scripts que se encuentran en la carpeta js.

/FRONTEND/juego/img



La carpeta img contiene todos los archivos como imágenes, sonidos, música, sprites, escenarios e iconos que requiere el juego para poder ejecutarse.


A continuación hay una descripción de cada subcarpeta de la carpeta IMG.

/FRONTEND/juego/img/Componentes



En esta carpeta se encuentran todas las imágenes y sonidos que requiere el juego según el módulo que lo necesite.

/FRONTEND/juego/img/escenarios



Aquí se encuentran las imágenes de los escenarios que se ven a lo largo de los mundos del juego.

/FRONTEND/juego/img/IconoAplicacion



El archivo que se encuentra en esta carpeta es el icono de la aplicación para el archivo html

/FRONTEND/juego/img/personajes



Esta carpeta contiene las imágenes, sprites, poderes y botones de los personajes del Pacto de Honor Grancolombiano.


Así mismo, están los archivos correspondientes de los mini personajes del juego.

/FRONTEND/juego/java



Esta carpeta contiene archivos necesarios para realizar el Login del juego

/FRONTEND/juego/js



La carpeta JS contiene los scripts del juego.


Cada subcarpeta contiene un archivo con extensión .js (Javascript)

Para modificar algo en el juego se debe navegar en esta carpeta, a través de la subcarpeta correspondiente al cambio que se desee hacer.

/FRONTEND/juego/Jsp



Contiene los archivos JSP requeridos para el Login del juego.

/FRONTEND/portal



La carpeta portal contiene archivos necesarios para la ejecución del portal del juego.
